## Instructions

### Run locally
- Install [Yarn](https://yarnpkg.com/pt-BR/docs/install)
- Install [Vue-cli (beta version)](https://github.com/vuejs/vue-cli) with Yarn: run `yarn global add @vue/cli`
- Run `yarn install`
- Run `yarn serve`

### Lint
- Run `yarn lint [--fix]`

### Build
- Run `yarn build`
